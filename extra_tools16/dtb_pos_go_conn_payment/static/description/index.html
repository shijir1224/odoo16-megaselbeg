ePOS Plugin

 

Pass Plugin нь Odoo ERP системийн ПОС-ийн програм дээр Pass ухаалаг түрийвчээр төлбөр төлөх боломж олгоно. Pass нь Монгол улсын төлбөр тооцооны аюулгүй байдлын шаардлагыг хангасан, оюуны өмчөөр баталгаажсан бүтээгдэхүүн бөгөөд бүх төрлийн төлбөр тооцоог банкны карт болон бэлэн мөнгө ашиглахгүй дижитал хэлбэрээр хийх ухаалаг шийдэл юм.

 

Давуу тал

  • Төлбөр, картын мэдээллийн аюулгүй байдлын PCI DSS LEVEL 1 стандартыг хангасан
  • Magstripe
  • ICC /Чип/
  • Contactless
  • PASS QR code /MPM/

 

Шийдэл 

MPM (Merchant Presented Mode)

Мерчантын үүсгэсэн QR болон Bar code-ийг Pass апп-аар уншуулж төлбөр төлнө.

Дэлгэрэнгүй мэдээлэл: DATABANK.MN

 

Суулгах, Тохиргоо хийх заавар

 

1. Apps дотроос Pass түлхүүр үгээр хайж install хийнэ.

 

2. POS-ийн тохиргоонд байх Payment methods цэс дотор PASS гэсэн нэртэй төлбөрийн төрөл нэмэгдсэн байна. Дээр нь дарж ороод нэмэлт дараах тохиргоог хийнэ:

  • 'Journal' хэсэг банкны журнал сонгоно
  • 'Use a Payment' хэсэгт 'PASS payment' сонгогдсон байх ёстой
  • 'Active pass test mode' хэсгийг та сонгосон бол PASS-гүйлгээг ямар нэгэн token-гүй тестийн гүйлгээ хийх боломжтой болно. Хэрэв бодитоор ашиглах бол энэ хэсгийг сонгохгүй байх ёстой

 

3. Хэрэв production орчинд ашиглах бол 'Pass Ecommerce Token' хэсэгт DATABANK ХХК-аас өгсөн token-г бөглөж PASS-ын төлбөрийг ашиглах боломжтой. System settings дотор тохируулна

 

 

Хэрэглэх заавар

 

1. Захиалгаа бичээд төлбөр хийх хэсэг 'PASS' дээр 'Send' дарахад уншуулах QR код гарч ирнэ

 

2. Төлбөр хийгдсэний дараа QR цонх хаагдаж 'Payment successful' гэсэн төлөв орж VALIDATE товч идэвхижиж захиалга дуусгах боломжтой болно
